# Salubrum Projects Portfolio

## Growth Strategy Work at Salubrum

This portfolio showcases comprehensive growth strategy work conducted at Salubrum, an AI healthcare startup focused on leveraging data analytics to drive strategic decisions and market growth.

### Project Overview

#### 1. DH Audience Analytics (15 Procedures)
- Analyzed 300M+ consumer health records across Definitive Healthcare database
- Developed filter criteria for 15 surgical procedures with exact DH propensity codes
- Created audience segmentation by demographics, clinical propensity, and economic indicators
- Deliverable: Complete procedure-level audience counts and targeting documentation

#### 2. Premium Care Plastic Surgery Strategy
- Developed 5 distinct personas for premium cosmetic procedures (Mommy Makeover, Tummy Tuck, BBL, Facelift, Breast Augmentation)
- Created economic targeting logic specific to cash-pay, value-conscious consumers
- Built conversion hypothesis framework for each persona
- Documented StackAdapt campaign specifications and messaging strategies

#### 3. Healthcare Market Analysis
- Analyzed medical tourism trends and economic drivers
- Evaluated GLP-1 adoption patterns and demographic shifts
- Assessed competitive landscape for AI healthcare solutions
- Identified growth opportunities in underserved segments

#### 4. SQL Analytics & Query Templates
- Developed reusable SQL queries for DH audience extraction
- Built demographic segmentation and propensity join patterns
- Created templates for healthcare analytics workflows
- Documented best practices for large-scale health data analysis

#### 5. Dashboard & Campaign Analytics
- Designed Tableau dashboards for real-time campaign performance tracking
- Created StackAdapt ad specifications (300x250, 300x600, 728x90, Leaderboard)
- Developed budget allocation and testing prioritization frameworks
- Built messaging customization by persona and channel
